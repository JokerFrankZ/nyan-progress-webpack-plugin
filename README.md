## Install

```
$ npm i -D nyan-progress-webpack-plugin
```

## Usage

Add this to your webpack plugin:

```
var NyanProgressPlugin = require('nyan-progress-webpack-plugin');

...

plugins: [
  new NyanProgressPlugin()  
]

...
```
